# HealSync

HealSync is a healthcare emergency assistance platform designed to provide instant medical information through QR scanning.

## Features

- Emergency QR medical profile
- Medicine reminders
- Medical report storage
- OCR prescription reader
- Hospital finder with Google Maps
- Health expense tracker
- Insurance guidance

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Firebase
- Google Maps API

## Running the Project

Install dependencies:

npm install

Start the development server:

npm run dev

Then open:

http://localhost:5173
